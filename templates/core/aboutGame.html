O Jogo

O Jogo da bolsa é um ambiente de simulação de investimento na bolsa de valores de São Paulo (B3), e tem o objetivo de ser um laboratório de prática de programação e de tecnologia. Seu desenvolvimento base é na linguagem python com a utilização do framework Django. O código fonte do projeto é encontrado no GitHub.
